# LCM-HS-Imager-Hyperspectral-Imaging-Code
The pre-trained deep neural network was exported in the ONNX (Open Neural Network Exchange) format and integrated into the MATLAB environment for inference. Leveraging MATLAB’s deep learning toolbox, the ONNX model was imported to perform spectral reconstruction, enabling a seamless workflow between the neural network and numerical optimization.
Please ensure the Deep Learning Toolbox Converter for ONNX Model Format is installed in MATLAB to load the .onnx model.
The full reconstruction code, pre-trained ONNX models, and raw measurement data supporting the results in Figure 4 and Figure 5 are available in the Zenodo repository at https://doi.org/10.5281/zenodo.19583468.
